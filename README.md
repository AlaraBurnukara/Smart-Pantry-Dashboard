# Smart Pantry #

Smart Pantry is a sustainable food management web application designed to help users reduce food waste and monitor pantry items efficiently.  
The project aligns with the **United Nations Sustainable Development Goal (SDG) “Zero Hunger”**, promoting awareness of food consumption and responsible waste management.

---

# Features #

- Interactive Dashboard** for tracking pantry statistics and user activity  
- Notification Preferences and Privacy & Security Settings
- Profile Edit Section** with “Save”, “Cancel” and “Delete Account” functionality  
- Recent Activity 
- Minimal, accessible design based on the Figma prototype
